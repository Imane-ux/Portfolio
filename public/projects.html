Yatzy Game

  • I created the Yatzy game using HTML, JavaScript, and CSS, ensuring an interactive user experience.
  • Player rolls five dice up to two times per turn to achieve specific combinations for points.
  • The objective is to fill all score boxes with the highest possible scores in categories like ones, twos, full house, and more.
  • Game ends when all the score boxes are used and a final tally is given.
  • Enhanced with PHP server component to store game state and leaderboard.
  • Updated JavaScript to make AJAX calls to the API for dynamic updates.
  • Player can play multiple rounds in a session and a game leaderboard will track the top 10 scores.

Hospital Triage Application

Medical Care is a web application designed to help hospital staff and patients better understand wait times in the emergency room.
It allows triage staff to administer the system based on severity of injuries and queue length.
It provides patients with approximate wait times via a sign-in feature.

Languages and Technologies Used:
Frontend:
HTML/CSS for design and layout.
JavaScript for client-side interactions and real-time updates.
Backend:
PHP for server-side processing and handling requests.
Database:
PostgreSQL for managing patient data and wait times.
